1. Download and Install jre 1.5 from http://java.sun.com/javase/downloads/index.jsp
   if java is not already installed.
2. Run bin\RunAstroSoft.bat
3. To change place of birth, type in city name and click search icon, Longitude/Latitude of place will be automatically populated.
   All the major cities in India is available with software. If it doesn't find city, just type the city name and longitude/latitude yourself.
4. You can also set Default Location (using menu options->Default Location) to make Horosocope, Panchang and Ephemeris use it as default.
